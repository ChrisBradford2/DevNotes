## A propos de DevNotes

DevNotes est une plateforme web faisant office de Wiki pour le bachelor 3 INGLOG du Campus Paris Ynov permettant de visualiser les notes de cours des étudiants.

### Construit avec

Le wiki et son contenu sont contenus dans [ce dépôt](https://github.com/ChrisBradford2/DevNotes), et le système a été construit en utilisant [Docusaurus v2.0](https://docusaurus.io/).

<!-- GETTING STARTED -->

## Premiers pas

### Prérequis

Installez [Node.js](https://nodejs.org/en/download/).

### Installation locale

1. Executez `yarn install` dans votre terminal.
2. Executez `npx docusaurus start`
3. Ouvrez la page `http://localhost:3000/DevNotes`

<!-- CONTRIBUTING -->

## Contribution

DevNotes est un développement communautaire et nous examinerons toutes les demandes d'ajout et les problèmes postés dans ce dépôt. Si vous remarquez des bugs, ou si vous pensez que nous avons oublié quelque chose, n'hésitez pas à créer une demande de modification avec vos suggestions. Les ajouts de contenu plus importants et les suggestions peuvent être considérés comme des problèmes qui peuvent ensuite être discutés avec l'équipe qui gère la page.  
Jetez un coup d'œil à [Contribution](/contrib).

Si vous souhaitez ajouter vos propres cours ou un article de blog :
1. Forkez le projet
2. Créez une branche Feature (`git checkout -b feature/AmazingFeature`)
3. Commitez vos changements (`git commit -m 'Add some AmazingFeature'`)
4. Pushez sur la branche (`git push origin feature/AmazingFeature`)
5. Ouvrez une Pull Request
